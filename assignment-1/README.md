# Assignment 1 - Landing Page Development (Plain HTML & CSS)

## Objective
Recreate a professional website landing page using only HTML5 and Plain CSS (no frameworks, no animations, no responsiveness).

## Reference Website
**Auto Care** - E-commerce website for automotive products

## Project Structure
```
assignment-1/
├── index.html          # Plain HTML file
├── styles.css          # Plain CSS stylesheet (no animations, no responsive)
└── README.md          # This file
```

## Technical Requirements ✅

### HTML5
- Semantic HTML5 structure
- No JavaScript
- No framework libraries

### Plain CSS Only
- No CSS animations or transitions
- No responsive design (media queries removed)
- No CSS frameworks
- Simple, static styling
- Button styles matching original project

## Features Implemented

### 📄 HTML Structure
1. **Navigation Bar** - Logo, links, cart, login
2. **Hero Section** - Title, tagline, CTA button
3. **Featured Products** - 4 products in grid
4. **Footer** - Copyright information

### 🎨 CSS Styling
- Fixed 4-column product grid
- Red color scheme (#dc143c, #8b0000, #ffebee)
- Simple button styles (no hover animations)
- Basic layout with no responsiveness
- Plain static styling

## Color Scheme
- Primary Red: #dc143c
- Dark Red: #8b0000
- Light Red: #ffebee
- Text Dark: #0f172a
- Text Muted: #64748b
- White: #ffffff

## Button Styles
- **Primary Button**: White background with red text
- **Secondary Button**: Red background with white text
- No hover animations or transitions

## Product Cards
- 4-column grid layout (fixed, non-responsive)
- Image, category badge, title, description, price, button
- Category badges with light red background
