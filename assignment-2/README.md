# Assignment 2: Responsive Top Menu (JS)

## Objective
Enhance the existing project by making the top navigation menu fully responsive using plain CSS and Vanilla JavaScript (no frameworks).

## Features
- CSS media queries for layout adjustment
- Desktop: Horizontal navigation menu
- Mobile/Tablet: Collapses into a hamburger (☰) icon
- Vanilla JS toggle for smooth menu opening/closing

## How to Run
Simply open `index.html` in any modern web browser. Shrink the browser width to see the hamburger menu and click it to toggle.
