# Lab Task 2: Express Based Application

## Objective
Convert the static landing page into an Express-based application using EJS templating.

## Features
- Node.js & Express.js backend
- EJS template engine (`express-ejs-layouts`)
- Serves the responsive landing page via Express routes
- Responsive navigation (from A2 and LA1)
- No database or authentication integrated yet

## Setup Instructions
1. Ensure Node.js is installed.
2. Open terminal in this directory.
3. Run `npm install` to install dependencies (express, ejs, express-ejs-layouts, etc).
4. Run `node server.js` to start the server.
5. Visit `http://localhost:3000` in your browser.
