# Lab Task 1: Responsive Landing Page

## Objective
Improve the Assignment 1 Landing Page by making it fully responsive for mobile devices using plain CSS Media Queries.

## Features
- HTML5 structure
- Plain CSS styling
- Responsive layout adjustments using `@media` queries for mobile devices (max-width: 768px).

## How to Run
Simply open `index.html` in any modern web browser and resize the window to see the responsive layout.
